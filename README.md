# Iris Flower Classification

## Overview
This project is part of the CodeAlpha Data Science Internship.

The objective is to build a machine learning classification model that predicts the species of Iris flowers based on their measurements.

## Dataset
- Iris Flower Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Workflow
1. Load the dataset
2. Explore the data
3. Split the dataset into training and testing sets
4. Train the classification model
5. Predict flower species
6. Evaluate model accuracy

## Output
The trained model predicts one of the following Iris species:
- Setosa
- Versicolor
- Virginica
